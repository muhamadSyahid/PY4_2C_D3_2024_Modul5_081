## Lesson Learnt

Resolusi kamera yang lebih tinggi seperti `max` atau `ultraHigh` memang memberi detail visual yang lebih tajam, tetapi beban pemrosesan, panas perangkat, dan latensi preview ikut naik. Untuk aplikasi patroli jalan raya, detail ekstra sering tidak sebanding dengan penurunan FPS dan risiko frame drop, jadi preset sedang biasanya lebih stabil untuk alur deteksi real-time.
